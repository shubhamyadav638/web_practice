Float left

HTML stands for Hyper Text Markup Language. It’s like the scaffolding that holds up the digital architecture of web pages. When you visit a website, your browser (whether it’s Chrome, Edge, Firefox, or Safari) reads the HTML documents and uses them to display everything correctly. But here’s the cool part: browsers don’t actually show you the HTML tags; they just use them to figure out how to present the content. CSS stands for Cascading Style Sheets. It’s like the fashion designer of the web world. When combined with HTML (which you just learned about), CSS gives your web pages that snazzy makeover they deserve.
